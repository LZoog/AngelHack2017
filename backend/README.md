# AngelHack2017 Backend

# Setup
```sh
npm run database
```

```sh
npm start
```

# Requirements
  * [NodeJS](https://nodejs.org/)
  * [Docker](https://www.docker.com/)
  * [Alexa](https://aws.amazon.com/awis/)

# Deployment

Automatically builds and pushes to Docker when committed to master branch

```sh
npm run build
npm run push
```
